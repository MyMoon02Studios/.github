# MyMoon — Plataforma de Chat Web

Bienvenido a **MyMoon**, una innovadora plataforma de chat web que conecta personas a través de conversaciones seguras, rápidas y amigables.

---

## Nuestra misión  
Ayudar a las personas a comunicarse de manera fluida, privada y divertida, sin importar dónde estén.  
**MyMoon** busca ofrecer una experiencia de chat moderna, confiable y accesible desde cualquier navegador.

---

## Quiénes somos  
- **Nombre:** MyMoon  
- **Visión:** Convertirnos en la referencia de aplicaciones de chat web para comunidades, equipos y usuarios que valoran usabilidad y privacidad.  
- **Valores:**  
  - **Privacidad:** Prioridad absoluta en la protección de datos.  
  - **Simplicidad:** Interfaz limpia, intuitiva y sin barreras.  
  - **Velocidad:** Respuestas en tiempo real y mínimo tiempo de espera.  
  - **Innovación abierta:** Escuchamos a nuestros usuarios y adaptamos la experiencia.

---

## Tecnologías que utilizamos  
- **Frontend:** React, Next.js, Tailwind CSS  
- **Backend:** Node.js, Express / NestJS  
- **Base de datos:** PostgreSQL, Redis (para sesiones y caché)  
- **Comunicación en tiempo real:** WebSockets (Socket.io)  
- **Infraestructura / DevOps:** Docker, Kubernetes, GitHub Actions, AWS / GCP  
- **Autenticación / Seguridad:** OAuth2, JWT, cifrado de datos en tránsito y en reposo  

---

## Funcionalidades clave  
✅ Registro y autenticación segura  
✅ Chats uno-a-uno y chats de grupo  
✅ Envío de mensajes de texto, emojis, archivos e imágenes  
✅ Indicadores de “escribiendo”, “visto” y “en línea”  
✅ Notificaciones en tiempo real  
✅ Historial de mensajes con búsqueda  
✅ Ajustes de usuario (perfil, privacidad, estado)  
✅ Diseño responsive (móvil y escritorio)  
✅ Moderación básica y control de abuso
